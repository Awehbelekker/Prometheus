================================================================================
  WHAT EACH SETTING MEANS
================================================================================

"Enable ActiveX and Socket Clients"
   - This ENABLES the API
   - Without this, NO programs can connect
   - MUST be checked ✅

"Read-Only API"
   - If checked: Can only READ data (quotes, positions)
   - If unchecked: Can PLACE ORDERS and trade
   - We need to PLACE ORDERS, so UNCHECK this ❌

"Socket port"
   - 7496 = Live trading (real money)
   - 7497 = Paper trading (simulated)
   - We want LIVE trading, so use 7496

"Trusted IP Addresses"
   - List of IPs that can connect
   - 127.0.0.1 = Your local computer
   - This allows our bot to connect
